# Nexify Fullstack\n\n- Booking with deposit (Stripe)\n- Supabase persistence\n- Portal, Support Chat, Personal Assistant\n\n## Run\n```bash\nnpm i\ncp .env.example .env\nnpm run dev\n```\nOpen http://localhost:3000 → /demo, /portal, /support, /assistant\n\n## Supabase\nRun SQL in `supabase.sql`. Set `NEXT_PUBLIC_SUPABASE_URL` and `SUPABASE_SERVICE_ROLE`.\n\n## Stripe\nSet `STRIPE_SECRET_KEY`. Webhook: `stripe listen --forward-to localhost:3000/api/stripe/webhook` → put signing secret in `STRIPE_WEBHOOK_SECRET`.\n\n## OpenAI\nSet `OPENAI_API_KEY` and wire the API routes (commented sections).\n\n## Deploy\nPush to GitHub → Import to Vercel → add envs → Deploy.\n
